# Slicetime

## Getting Started

Clone the repository from below link

```sh
https://github.com/99x-academy/slicetime.git
```

Go into the project folder

```sh
cd slicetime
```

Install npm

```sh
npm i
```

Run the project

```sh
npm start
```
